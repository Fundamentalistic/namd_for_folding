WorkDistrib

Messages

MapDistribMsg
(packed) Contains PatchMap and ComputeMap, used at startup.
ComputeMapDistribMsg
(packed) Only ComputeMap, used for load rebalancing.

Entry Points

WorkDistrib(InitMsg *)
Initialization, no external effects.
saveMaps(MapDistribMsg *)
Curious little function. The maps actually take care of themselves in their unpack methods so this just initializes the node pointer and sets a flag.
recvComputeMap(ComputeMapDistribMsg *)
ComputeMap unpacks itself, but since this is used for rebalancing it sends a DoneMsg to doneSaveComputeMap on P.E. 0.
doneSaveComputeMap(DoneMsg *)
Counts down nodes, then sends a DoneMsg to the entry point set in saveComputeMap.
enqueueWork(LocalWorkMsg *)
Receives LocalWorkMsg sent in messageEnqueueWork. Acts as a queueing system and calls doWork on the compute object stored in the message. enqueueWork[ABC] have been added to aid analysis.
