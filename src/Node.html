Entry Points

Node(GroupInitMsg *)
Initialization, binds other B.O.C.'s.
startUp(QuiescenceMessage *)
Broadcasts InitMsg to startup after quiescence.
startup(InitMsg *)
Consists of a case statement and counter to trigger various stages of startup. Turns on quiescence detection with message targeted at startUp on P.E. 0. After last stage broadcasts RunMsg to run via messageRun.
quiescence(QuiescenceMessage *)
Terminates program - used for development.
run(RunMsg *)
Runs simulation after startup barriers by starting Controller and Sequencers.
homeDone(DoneMsg *)
Counts down HomePatch completions and sends DoneMsg to nodeDone on P.E. 0 when finished.
nodeDone(DoneMsg *)
Counts down Node completions. Calls Namd::namdDone when finished.
