ComputeMgr

Messages

Entry Points

ComputeMgr(InitMsg *)
Initialization only.
updateComputes2(QuiescenceMessage *)
Triggered by quiescence set on P.E. 0 in updateComputes to let all calculations finish. Call WorkDistrib::saveComputeMap to get new compute map and then message updateComputes3.
updateComputes3(DoneMsg *)
Broadcasts RunMsg to updateLocalComputes.
updateLocalComputes(RunMsg *)
Prepares to create or delete Computes based on map. Sets quiescence on P.E. 0 to updateLocalComputes2.
updateLocalComputes2(QuiescenceMessage *)
Triggered by quiescence set on P.E. 0 in updateLocalComputes. Broadcasts RunMsg to updateLocalComputes3.
updateLocalComputes3(RunMsg *)
Actually creates or deletes computes. Sets quiescence on P.E. 0 to doneUpdateLocalComputes.
doneUpdateLocalComputes(DoneMsg *)
Triggered by quiescence set on P.E. 0 in updateLocalComputes3. Sends DoneMsg to entry point originally passed to updateComputes.
