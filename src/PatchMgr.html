PatchMgr

Messages

MovePatchesMsg
(packed) Contains basic information about all atoms in a patch.
MigrateAtomsMsg
(packed) Contains a MigrationList with total information about all atoms to be moved.
MigrateAtomsCombinedMsg
(packed) Same as MigrateAtomsMsg only data for different patches is mixed to reduce message count. Has distribute method to deposit data to patches.

Entry Points

PatchMgr(InitMsg *)
Checks singleton status. Instances and binds to PatchMap. Initializes migration message combining system.
recvMovePatches(MovePatchesMsg *)
Creates a home patch based on data sent by sendMovePatches.
recvMigrateAtoms(MigrateAtomsMsg *)
Deposits data sent by sendMigrationMsg to appropriate patch.
recvMigrateAtomsCombined(MigrateAtomsCombinedMsg *)
Calls message to distribute combined data sent by sendMigrationMsgs.
