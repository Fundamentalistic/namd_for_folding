ProxyMgr

Messages

RegisterProxyMsg
Contains node and patch IDs.
UnregisterProxyMsg
Contains node and patch IDs
ProxyAtomsMsg
(packed) Contains AtomIDs but not positions.
ProxyDataMsg
(packed) Contains Positions but not AtomIDs.
ProxyAllMsg
(packed) Contains AtomIDs and Positions.
ProxyResultMsg
(packed) Contains Forces.

Entry Points

ProxyMgr(InitMsg *)
Checks singleton status.
recvRegisterProxy(RegisterProxyMsg *)
Passes to HomePatch::registerProxy.
recvProxyData(ProxyDataMsg *)
Deposits data sent by home patch via sendProxyData to proxy. Used during normal timesteps.
recvProxyAtoms(ProxyAtomsMsg *)
Deposits data sent by home patch via sendProxyAtoms to proxy. Used when a proxy is created, when sendProxyAtoms is called by HomePatch::registerProxy.
recvProxyAll(ProxyAllMsg *)
Deposits data sent by home patch via sendProxyAtoms to proxy. Used during atom migration timesteps.
recvResults(ProxyResultMsg *)
Deposits data sent by proxy via sendResults to home patch.
