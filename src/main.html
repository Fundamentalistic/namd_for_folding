Entry Points

main()
Initial Charm startup entry point, only called on PE 0. Creates a Namd instance (constructor generates all B.O.C.'s), checks arguments, and calls Namd::startup (initializes NamdState from config file and broadcasts InitMsg to Node::startup via Node::messageStartUp).
